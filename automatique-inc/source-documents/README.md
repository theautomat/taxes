# Automatique Inc - Source Documents

Place source documents in the appropriate folders:

## Folder Structure

```
source-documents/
├── mercury/           # Mercury bank statements and CSV exports
├── stripe-atlas/      # Incorporation docs, EIN, bylaws, cap table
├── privacy-com/       # Privacy.com transaction exports (CSV preferred)
├── wells-fargo/       # Personal Wells Fargo transactions related to Automatique
└── contractors/       # Contractor agreements, payment records, 1099 info
```

## Preferred Formats

- **Bank transactions:** CSV exports (easier to process than PDFs)
- **Legal documents:** PDF
- **Statements:** PDF for reference, CSV for processing

## Naming Convention

Use descriptive names with year:
- `2024_mercury_transactions.csv`
- `2025_mercury_statements_jan-dec.pdf`
- `2024_privacy-com_transactions.csv`
- `stripe-atlas_certificate-of-incorporation.pdf`
